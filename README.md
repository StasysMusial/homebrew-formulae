# StasysMusial Formulae

## How do I install these formulae?

`brew install stasysmusial/formulae/<formula>`

Or `brew tap stasysmusial/formulae` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "stasysmusial/formulae"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
